# Vizitková stránka (ukázka)

Otevři soubor `index.html` v prohlížeči, například:

```bash
open index.html
```

Obsahuje: jméno, email, telefon, adresu a krátké bio.

Funkce navíc:

- Tlačítka pro kopírování emailu a telefonu.
- Možnost stáhnout kontaktní vCard (`contact.vcf`).

Úprava údajů: otevři `index.html` v textovém editoru a změň texty nebo `contact.vcf` pro vCard.
